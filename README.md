# MyDish

**My dish** es una aplicación web diseñana para la creación de platos saludables siguiendo el formato de _"Plato hardvard"_ o plato saludable el cual se compone de un 50% de vegetales, un 25% de proteínas y un 25% de carbohidratos saludables.

La aplicación permite que una vez que el usuario se registre e inicie sesión, elija entre una variedad de alimentos, uno de cada tipo para conformar su plato, pudiendo, cambiar el alimento, darle nombre al plato, guardarlo e incluso generar un plato con alimentos aleatorios. 

## Primeros pasos
El primer paso para utilizar MyDish es clonar el repositorio.
Puedes hacerlo de esta forma:
```
git clone https://github.com/Atibrox95/mydish-Front.git 
```
Una vez clonado entras dentro de la carpeta de MyDish e instalas sus dependencias. 
Puedes hacerlo de esta forma:  
```
npm install
```
Una vez finalizada la instalación ya puedes empezar a usar MyDish ejecutanto el siguiente comando:
```
npm run dev
```
### Nota:
Para que Mydish funcione correctamente es necesario arrancar el servidor, las instrucciones para hacerlo las puedes encontrar [aquí](https://github.com/Atibrox95/mydish.git)
